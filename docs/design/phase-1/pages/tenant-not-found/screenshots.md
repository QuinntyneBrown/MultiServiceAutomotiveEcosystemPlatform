# Screenshots - Tenant Not Found Page

## How to View Mockups

Open `mockup.html` in any modern browser and use browser DevTools to simulate different viewport sizes:

### Mobile View (375px x 812px)
1. Open `mockup.html` in browser
2. Open DevTools (F12 or Cmd+Option+I)
3. Toggle Device Toolbar (Ctrl+Shift+M or Cmd+Shift+M)
4. Select iPhone 12/13 or set custom size: 375 x 812

### Tablet View (768px x 1024px)
1. Open `mockup.html` in browser
2. Open DevTools
3. Toggle Device Toolbar
4. Select iPad or set custom size: 768 x 1024

### Desktop View (1440px x 900px)
1. Open `mockup.html` in browser
2. Open DevTools
3. Toggle Device Toolbar
4. Set custom size: 1440 x 900

## Generating Actual Screenshots

To generate PNG screenshots, you can use any of these methods:

### Method 1: Browser DevTools
1. Open mockup.html in Chrome
2. Open DevTools â†’ Toggle Device Toolbar
3. Set viewport size
4. Right-click â†’ Capture screenshot

### Method 2: Playwright/Puppeteer Script
```javascript
const { chromium } = require('playwright');

async function captureScreenshots() {
  const browser = await chromium.launch();
  const page = await browser.newPage();

  const viewports = [
    { name: 'mobile', width: 375, height: 812 },
    { name: 'tablet', width: 768, height: 1024 },
    { name: 'desktop', width: 1440, height: 900 }
  ];

  for (const vp of viewports) {
    await page.setViewportSize(vp);
    await page.goto('file:///path/to/mockup.html');
    await page.screenshot({
      path: `screenshot-${vp.name}.png`,
      fullPage: true
    });
  }

  await browser.close();
}
```
