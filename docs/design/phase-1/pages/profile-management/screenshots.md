## Interaction States Summary

### Form Field States
1. **Default**: Gray border, placeholder text
2. **Focus**: Blue border, blue glow
3. **Filled**: Black text on white
4. **Error**: Red border, error icon and message
5. **Success**: Green border (for validated fields like email)
6. **Disabled**: Gray background, gray text

### Button States
1. **Primary**: Blue background (#00529F)
2. **Primary Hover**: Darker blue (#003366)
3. **Primary Disabled**: 60% opacity
4. **Secondary**: White background, blue border
5. **Secondary Hover**: Light blue background
6. **Ghost**: Transparent, gray text
7. **Ghost Hover**: Light gray background

### Tab States
1. **Inactive**: Gray text, no underline
2. **Hover**: Light blue background
3. **Active**: Blue text, blue bottom border
4. **Complete**: Green dot indicator
5. **Incomplete**: Yellow dot indicator
6. **Empty**: No indicator

### Upload States
1. **Empty**: Dashed border, upload icon
2. **Hover**: Blue border
3. **Uploading**: Progress bar
4. **Uploaded**: Image preview with edit/delete buttons
5. **Error**: Red border, error message

---

## Responsive Breakpoints

### Desktop (> 992px)
- Sidebar visible
- Two-column form layout
- Full-width image previews
- Horizontal tab navigation

### Tablet (576px - 991px)
- Sidebar hidden (hamburger menu)
- Two-column form layout (some sections)
- Smaller image previews
- Horizontal scrolling tabs

### Mobile (< 576px)
- No sidebar
- Single-column layout
- Stacked buttons
- Scrollable tabs
- Full-width elements
- Bottom action buttons

---

## Accessibility Features

1. **Keyboard Navigation**
   - Tab through all form fields
   - Arrow keys in dropdowns
   - Escape to close modals
   - Enter to submit

2. **Screen Reader Support**
   - All images have alt text
   - Form labels properly associated
   - Error messages announced
   - Status updates announced

3. **Visual Indicators**
   - Focus outlines visible
   - Error states have icons
   - Success states confirmed
   - Loading states shown

4. **Color Contrast**
   - Text meets WCAG AA standards
   - Error red is accessible
   - Link blue is accessible
   - All buttons have sufficient contrast
