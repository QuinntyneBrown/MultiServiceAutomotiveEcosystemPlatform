## Component States

### Empty State (New Customer)
- All fields blank except defaults
- Language defaults to "English"
- Email, SMS checkboxes checked by default
- Phone call checkbox unchecked
- Marketing consent unchecked

### Filled State (Edit Customer)
- All fields populated with existing data
- Tags displayed as chips
- Success indicators on valid fields

### Error State
- Red border on invalid fields
- Error icon (ğŸ”´) next to field label
- Error message below field
- First error field focused on submit

### Loading State
- Submit button shows spinner
- Form fields disabled during submission
- "Saving..." text on button

### Success State
- Success notification appears
- Redirect to customer detail (default)
- OR form resets if "Save and add another" checked

---

## Interaction States

### Focus States
- Blue outline (shadow-focus) on focused input
- Increased border width
- Clear visual feedback

### Hover States
- Cancel button: light gray background
- Save button: darker blue background
- Tag remove buttons: darker background

### Disabled States
- Gray background on disabled inputs
- Lower opacity
- No cursor on hover
