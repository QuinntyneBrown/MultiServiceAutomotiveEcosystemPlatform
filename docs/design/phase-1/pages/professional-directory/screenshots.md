## Animation & Transitions

### Page Load
1. Hero section fades in (300ms)
2. Search bar slides up (200ms delay)
3. Filters fade in left (100ms delay)
4. Cards stagger in (50ms each, grid order)

### Filter Change
1. Results fade out (150ms)
2. Loading skeleton appears
3. New results fade in (300ms)
4. Count updates with highlight animation

### Card Interactions
- Hover: transform translateY(-2px) + shadow (200ms ease)
- Click: scale(0.98) for 100ms, then navigate

## Notes for Implementation

1. **Responsive Images**: Use `srcset` for profile photos at 1x, 2x resolutions
2. **Lazy Loading**: Implement `loading="lazy"` for images below fold
3. **Infinite Scroll**: Consider as alternative to pagination for mobile
4. **Touch Targets**: Minimum 44x44px for all interactive elements on mobile
5. **Focus Trap**: Implement for filter modal on mobile
6. **Skeleton Screens**: Show during initial load and pagination
7. **Error States**: Display inline errors for failed searches
8. **Debouncing**: Implement 300ms debounce on search input
